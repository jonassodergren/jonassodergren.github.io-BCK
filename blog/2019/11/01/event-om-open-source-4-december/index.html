Event om open source 4 december

Ett event om nyttan med öppen källkod
Friday, November 01, 2019

Den 4 december berättar Johan Linåker, Post Doc Researcher på Lunds Universitet, om sin forskning kring open source, ekosystem och Government-as-a-Platform. Han kommer att djupdyka i öppen källkod och:

varför offentlig sektor delar med sig jämfört med privat sektor vilka kostnader och risker som finns fördelar och drivkraft som skapas Han kommer att exemplifiera sina studier med JobTech Dev (Arbetsförmedlingens initiativ till utveckling av digital infrastruktur) både som plattform och som ekosystem.

Avslutningsvis ställer han sig frågan:

Kan konceptet JobTech Dev generaliseras bortom Arbetsförmedlingen?

Anmälning sker hos (Redpill)[https://www.redpill-linpro.com/events/offentlig-sektor-i-forandring]
