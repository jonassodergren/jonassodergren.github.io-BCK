Öppna Data

Enkel tillgång till data!

Våra API:er består av data som behövs för att matcha kompetens och efterfrågan på arbetsmarknaden. Jobtech strävar efter att leverera öppna data enligt sätt som möter användarens behov. En forskare kan hantera rådata, medan en användare som vill utveckla en digital tjänst efterfrågar strukturerad data. I vissa fall finns önskemål om berikad data såsom text enrichment och geodata på annonserna.

De olika leveranssätten för data:

  1. Data som fil: (exempelvis Jobstream och Ontologin)
  2. Data som API: (Jobsearch och Taxonomi)
  3. Exempeltjänst för att visa hur olika dataset kan användas i olika sammanhang (Jobscannern, Jobbometern).
  4. En fjärde nivå är widgets. Inget fristående system, men en funktion som tillgängliggör ett dataset i en slutanvändartjänst.

Annonser

Annonser som publiceras i Platsbanken tillhandahålls som API med en snabb och modern sökfunktion som har en marknadsledande hög träffsäkerhet på relevans samt är förlåtande då det kommer till felstavningar. En medveten satsning för att inte missgynna dem som har problem med stavning och språk. Annonser finns också som bulk-fil Vidare pågår ett arbete med att i offentlig och privat samverkan samla in alla digitala annonser på arbetsmarknaden och inte bara de annonser som inkommer till Arbetsförmedlingen. Projektet som har levererat ett MVP i form av ett API med annonser från närmare 300 olika annonssajter kallas Jobtech Jobs.

Taxonomi

Skulle matchning på arbetsmarknaden bli lättare om människor och system skulle kunna förstå varandra? Taxonomin innehåller begrepp som beskriver bland annat yrken, färdigheter och förhållanden mellan dem, som kan användas till exempel som bas för matchning. Innehållet uppdateras kontinuerligt i samråd med aktörer på arbetsmarknaden. Anslut er till vår API och få ert system att tolka CV och annonser för en bättre matchning.

Ontologi fil

Erfar ni att titlar, roller och kompetenser på arbetsmarknaden snabbt blir förlegade eller att ni inte hinner hitta stöd i era system för alla nya förmågor? Vi har försökt att råda bot på problemet. Med stöd av machine learning har vi analyserat alla annonser i Platsbanken de senaste 10 åren. Det rör sig och 6,5 miljoner annonser för att skapa en avancerad ordlista som innehåller de faktiska kompetenser, yrkestitlar och förmågor som står i annonser. Filen kommer under 2020 att integreras i Taxonomin, men för early adopters finns data tillgängligt redan nu.

My Data

Det finns ett par hundra sajter för att registrera sin CV. Tänk om det var möjligt att endast registrera sig en gång på ett ställe och att systemen samtidigt pratar med varandra! Ur detta perspektiv undersöker Digital Matchning Mydata-principerna och möjligheter med Egna utrymmet som syftar till interoperabilitet, portabilitet och individens rätt att bestämma vad man vill göra med sin data. Målet med arbetet hösten 2019 är att olika system på arbetsmarknaden skall kunna utbyta CV-data under förutsättningar att individen vill exportera och importera sin information mellan systemen. I ett större sammanhang kan tekniken/infrastrukturen användas för att olika myndigheter ska kunna utbyta grunddata sinsemellan och på så sätt effektivisera och förbättra de kundresor som spänner över myndighetsgränser.
